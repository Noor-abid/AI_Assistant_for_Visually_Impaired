5.3  Object  Guidance  Algorithm  1.  User  states  target  object  via  voice  (Web  Speech  API);  browser  streams  frames  at  10–15  FPS  via  
WebSocket.
 2.  YOLOv8  detects  target;  backend  computes  Euclidean  distance  to  frame  centre.  3.  Distance  mapped  to  haptic  pulse:  slow  (d  >  0.4),  medium  (0.2–0.4),  rapid  (d  <  0.2),  locked  (d  <  
0.05).
 4.  Browser  Vibration  API  +  audio  beeps  replicate  pulse;  TTS  announces  directional  cues  (Move  left,  
Tilt
 
up).
 YOLOv8  was  chosen  for  its  documented  0.23s  server-side  inference  and  80  pre-trained  COCO  classes.

8.  Statement  of  Limitations  •  Small  sample  (5–10  participants)  limits  statistical  generalisability.  •  Browser  Vibration  API  unsupported  on  iOS  Safari,  reducing  haptic  reliability  on  Apple  devices.  •  No  LLM  confidence  scoring;  hallucinated  outputs  may  not  be  flagged  to  users.  •  All  features  require  active  internet  —  offline/edge  AI  is  out  of  scope.
